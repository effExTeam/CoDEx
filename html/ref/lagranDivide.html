lagranDivide

lagranDivide[lagrangian,fieldList]
breaks the lagrangian into several parts. If fieldList contains n number of heavy fields, then the output is a List of n+1 elements. For the first n of those, the i th element is the part of the lagrangian with only terms involving the i th field. The last element of the output is comprised of the terms involving more than one heavy fields.

DetailsDetails

  • As the algorithm for determining Wilson Coefficients is different for single or multiple fields, lagranDivide is used by functions like treeOutput and loopOutput to treat different parts of the lagrangian separately.
