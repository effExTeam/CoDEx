CO D EX TUTORIAL

CoDEx Tutorial

CoDEx is a package which can be used to calculate Wilson Coefficients (both tree level and one loop) for a given BSM theory in which heavy degrees of freedom with spin 0, 1/2 or 1. The Wilson Coefficients will be evaluated at the scale of the mass of the heavy particle.

This package can also run the coefficients from the high scale (which typically is the mass of the heavy particle in the BSM theory) to weak scale or other low scales which user provides, keeping in mind the electroweak gauge symmetry is preserved in that scale.

In order to get the above mentioned outputs, user has to provide the representations of the heavy fields, the matrix structure of the symmetry generators and the BSM Lagrangian.

This package works on Mathematica 8 and 8+ versions.

Now, a mini window with title 

"How would you want to put your side of inputs here?"
will float on top of the Mathematica front end. The window will have three choices.

(1)I want to create my own "UserDefined.m" file now.
(2)I want to open a file from my computer.
(3)I want to use the "UserDefined.m" file already present in the current directory.
First time users are recommended to choose choice (1). This creates a file named "UserDefined.nb" in the NotebookDirectory[]. This file contains the demo inputs. One needs to edit these inputs according to the model in hand.
Choose choice (2) when an input tree (like the demo input) is already prepared.
Choose choice (3) when an input tree (like the demo input) is already prepared and is saved in the NotebookDirectory[].


Note the definition of hfvecs. Use hfvecs and the SM fields to define BSM Lagrangian.

Writing Lagrangian:

Once details of the heavy field is provided in the "UserDefined.m", the Mathematica Front End shows up the definition of the hfvecs. Use hfvecs and the SM fields to define BSM Lagrangian. Once again, a mini window with title 

"How would you want to define the Lagrangian?"
will float on top of the Mathematica front end. The window will have three choices.

(1)I want to write my own Lagrangian.
(2)I want to open a file from my computer.
(3)Help me write a Lagrangian.
First time users are recommended to choose choice (3). This shows a demo BSM Lagrangian and one can build one's own version of the BSM Lagrangian with similar construction.
Choose choice (2) when a Lagrangian (like the demo Lagrangian) is already prepared and you import it from a directory.
Choose choice (1) to write the BSM Lagrangian directly into Front End using the hfvecs and the SM fields.
