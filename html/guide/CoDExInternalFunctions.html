CO D EX PACKAGE GUIDE

CoDEx Internal Functions

This guide lists all the functions that CoDEx uses to do its job. Help files are not created for these functions and users are requested not to open these. Some of these we plan to make customizable later on, and their properties are listed in the next section, but for now, they are better left untouched.

ReferenceReference

smCoeff — picks the Standard Model Coefficient from the given expression.

fieldStructure — picks only the fields from the given expression

number — picks only the numbers from the given expression.

wcPick — finds the Wilson Coefficients from the given expression for a given operator.

byParts — takes expression1 D^{2} [ expression2 ] -> - D[ expression1 ] D[ expression2 ] where expression2 must be a series of field terms.

switchIntegral — takes expression1 D^{2} [ expression2 ] -> - D[ expression1 ] D[ expression2 ] but expression2 must be a single term.

filter — picks terms from an expression which have where i can takes values from 0 to 6.

bosonFieldGenerator — bosonFieldGenerator[B]= {B[0],B[1],B[2],B[3]}

hftlvecs — defines the solutions to the Euler-Lagrange equation for the provided lagrangian and heavy fields.

lagranKinMix — Checks whether Kinematic mixing is allowed or not.
