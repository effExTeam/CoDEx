Functions for building the Lagrangian

The user provides the Beyond Standard Model (BSM) Lagrangian for CoDEx to use. This is made up of the heavy multiplets and the Standard Model fields (See: SMFields). Other than these multiplets and fields, user may, and is asked to use the following functions while defining the BSM Lagrangian.

We do not provide separate help pages for these functions. This is why this guide displays a set of working examples in the last section.

FunctionsFunctions

re — re[expression] gives only the real part of expression.

im — im[expression] gives only the imaginary part of expression.

abs — Provides the norm of a vector. abs[A]= Sqrt[A.dag[A]]

dag — Stands for the Dagger operation on some expression.

dag[a]=(ComplexExpand//@(If[VectorQ[a]True,Conjugate[a],ConjugateTranspose[ExpandAll//@a]]))

commutator — commutator[a,b]=a.b - b.a (a and b should be matrices)

antiCommutator — anticommutator[a,b]=a.b + b.a (a and b should be matrices)

pau — Slightly modified version of PauliMatrix. pau[4]=*PauliMatrix[4]. Otherwise pau[i] = PauliMatrix[i].

tau — tau[a] = (1/2) * pau[a]

del — del is a variation of a partial derivative operator. In del[a,expression], a is the Lorentz index, and runs from 0 to 3.

hermitianConjugate — hermitianConjugate[a]=ComplexExpand[Conjugate[a]]

gamma — It is a user-defined matrix.

sigma — It is a user-defined matrix.

su3fijk — These are the SU(3) structure constants.
