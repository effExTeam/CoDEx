CoDEx

A practical two-step procedure of using the SMEFT to connect UV models of NP with weak scale precision observables.

basic working principle

The basic working principle of “CoDEx” is based on integrating out the heavy propagators using covariant derivative expansion (CDE) technique suitably blended within functional methods.

The WCs are computed in two different bases at the UV scale. There is a provision to perform the renormalisation group evolution of these Wilson coeffcients to generate the operators with rescpective WCs at the electro-weak scale. Here, we have chosen few BSM scenarios as example models to demonstrate how this code works. Using “CoDEx”, one can integrate out bosonic (both spin-0 and 1) and fermionic (spin-1/2) heavy degrees of freedom.

CoDEx is wrttien in Wolfram Language and contains detailed Mathematica-style documentation and sprawling tutorials. These are continually being updated as well.

Documentation

CoDEx documentation is fairly exhaustive and is integrated with Mathematica documentation. Successfull installation lets the user take advantage of it. The web-documentation mimics the same style and codes are ready to be directly copied. Clicking on the link above will take you there.

Parallel Computation

As CoDEx mainly handles large symbolic evaluations, most of the code is parallelized over available cores. This drastically reduces the evaluation time.
