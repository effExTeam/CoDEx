Documentation

CoDEx is wrttien in Wolfram Language and contains detailed Mathematica-style documentation and sprawling tutorials.
The web-documentation mimics the same style and codes are ready to be directly copied.

basic working principle

CoDEx  works by integrating out the heavy propagators using Covariant Derivative Expansion technique suitably blended within functional methods.
Wilson Coefficients are computed in two different bases at the UV scale. There is provision to perform the RG evolution of these WCs to electro-weak scale. Multiple example BSM scenarios are showcased to demonstrate the workflow of CoDEX.
