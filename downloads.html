Download & Install

CoDEx is developed using Wolfram language, keeping Mathematica notebook frontend in mind. Its documentation is detailed, fairly exhaustive, and created using Wolfram Workbench. It is thus fully integrable with inbuilt Mathematica documentation after installation. We here mention 2 different possible ways of installing CoDEx.

Method I Method II Contact Form

Method 1

The easiest, savviest way (Needs a live internet connection)

Step 1

Prepare

First, make sure you have Mathematica (R)  installed in your machine. Any version of mathematica from version 9 would work. Next, open a Mathematica Notebook, copy the following command there and run it**.

Import["https://github.com/effExTeam/CoDEx-1.0.0/raw/master/install.m"];
InstallCoDEx[]
 
**We have shamelessly modified the installer for FeynRules. Cheers to its developers!

Step 2

Work

If everything runs smoothly, CoDEX now runs in your machine and you'll able to see the welcome message. Here's the guide for first time use!

Method 2

If the previous method doesn't work, or you want to install it at your convenience, offline.

Step 1

Download archive

Download the .zip file from this link:
LINK 

Step 2

Unpack Archive

Next, unpack it with your favorite program to the mathematica "Applications" directory of you machine.
To do this from within Mathematica Kernel, copy the path of the downloaded archive (e.g, if you are working in windows and have downloaded the .zip file to the 'Downloads' folder, your path will be "C:\......\Downloads\CoDEx.zip". Let's call it 'path' from now), and run this command:
ExtractArchive['path', FileNameJoin[{$UserBaseDirectory,"Applications"}]]

Step 3

Work

If everything runs smoothly, CoDEX now runs in your machine and you'll able to see the welcome message. Here's the guide for first time use!
