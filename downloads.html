Download & Install

CoDEx is developed using Wolfram language, keeping Mathematica notebook frontend in mind. Its documentation is detailed, fairly exhaustive, and created using Wolfram Workbench. It is thus fully integrable with inbuilt Mathematica documentation after installation.

Install Downloads

~Installing~

Step 1

Prepare

First, make sure you have Mathematica (R)  installed in your machine. Any version of mathematica from version 9 would work. Next, open a Mathematica Notebook, copy the following command there and run it**.

Import["https://github.com/effExTeam/CoDEx-1.0.0/raw/master/install.m"];
InstallCoDEx[]
 
**We have shamelessly modified the installer for FeynRules. Cheers to its developers!

Alt-Step 1

Detailed Description

This page contains a detailed description of all possible installation procedures for CoDEx, in Mathematica Documentation format.

Step 2

Work

If everything runs smoothly, CoDEX now runs in your machine and you'll able to see the welcome message. Here's the guide for first time use!

If you're an user experienced with installing Mathematica packages, using the 'Install' option from the Notebook menu or manually extracting the downloaded archive file in the 'Applications' folder inside the $UserBaseDirectory works perfectly.
